# CandyWorld (WebGL版)

Unityで制作した一人称視点の探索ゲームです。

## 注意事項
* **動作保証**: WebGL版はブラウザの制限により、表示やフォントが正しく再現されない場合があります。
* **推奨**: 安定した動作を求める方は Windows版 をご利用ください。

## 開発環境
* **Unity**: 2021.3.45f1
* **レンダリングパイプライン**: URP (Universal Render Pipeline)
* **開発ツール**: Visual Studio 2022 / Visual Studio 2019
* **その他使用ソフト**: Blender / Scaniverse

## 操作方法
* **移動**: WASD
* **視点移動**: マウス
* **ジャンプ**: Space

## 使用素材クレジット
* **グラフィック**：いらすとや 様
* **BGM**：魔王魂 様
* **効果音**：効果音ラボ 様
* **システム**：Unity Starter Assets - First Person Character Controller
